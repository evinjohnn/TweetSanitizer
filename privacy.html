Privacy Policy for TweetSanitizer

Last Updated: November 25, 2025

1. Overview

TweetSanitizer("the Extension") is a browser extension designed to enhance the user experience on X (formerly Twitter) by displaying public location information next to usernames. We prioritize user privacy and data minimization.

2. Data Collection and Usage

The Extension performs the following actions:

  • Username Detection: It reads usernames visible on the webpage you are visiting (x.com or twitter.com).
  • Data Fetching: It makes direct HTTPS requests from your browser to Twitter's own GraphQL API (AboutAccountQuery) using your existing session.
  • Data Processing: It processes the "Location" field returned by X/Twitter to map it to a corresponding flag emoji.

3. Data Transmission and Storage

  • Local Caching: To reduce network requests, location data is cached locally in your browser using the chrome.storage.local API.
  • License Verification: If you purchase a Pro license, your license key is transmitted to our secure server (Cloudflare Workers) for validation. No other personal data is linked to this key.
  • Cloud Sync & Batch Lookups: To improve performance and reduce rate limits on X/Twitter, the extension uses a hybrid system. It may anonymously query our Cloudflare server for public location data of profiles you view. It may also contribute public location data found on your timeline to this shared cache to help other users.
  • No Personal Tracking: We do not track your browsing history, search queries, or private messages. We only process public profile information (usernames and location fields).

4. Permissions Justification

  • Host Permissions: Required to modify the DOM on x.com and twitter.com to insert flag emojis.
  • Storage: Required to cache location data locally.
  • Tabs/ActiveTab: Required to coordinate the on/off toggle in the popup menu.
